# Readme
## Installation Guide

### ZSH-Shell
```
wget https://raw.githubusercontent.com/prax93/linux-automation/refs/heads/main/configs/installer.sh -O ./.configGraber.sh

echo 'alias configupdater="sh ~/.configGraber.sh"' >> "$HOME/.zshrc"
soruce "$HOME/.zshrc"
```
